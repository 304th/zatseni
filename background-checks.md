# Background Checks SaaS: Deep Dive

Сервис проверки кандидатов для HR-отделов и служб безопасности

---

## Суть продукта

### Проблема

HR-отделы и службы безопасности проверяют кандидатов:
- **Вручную** — по 2-3 часа на человека, 10+ сайтов
- **Через дорогие enterprise-решения** — СПАРК (25k₽/мес), Контур.Фокус (50-70k₽/год)
- **Никак** — нет ресурсов, проверяют только топов

### Решение

API + Dashboard для мгновенной проверки по всем источникам:

```
HR загружает кандидата →
  система проверяет 10+ баз →
    отчёт за 2-5 минут →
      красные флаги подсвечены
```

### Что проверяем

| Источник | Данные |
|----------|--------|
| МВД | Розыск, недействительные паспорта |
| ФССП | Долги, исполнительные производства |
| ФНС | ИП, самозанятость, ИНН валидация |
| ЕГРЮЛ/ЕГРИП | Участие в бизнесе, аффилированность |
| Суды | Судимости, арбитражные дела |
| БКИ | Кредитная история (с согласия) |
| ГИБДД | Права, штрафы (для водителей) |
| Соцсети | OSINT анализ профилей |

> "Если количество кандидатов превышает 50 человек в месяц, компании жизненно важно автоматизировать сбор информации."

> "В 50% компаний задача проверки ложится на HR. Как правило, все негативные факторы проверить невозможно."

### Целевой сегмент

**Primary:**
- Компании 50-500 человек
- Найм 10-50 человек/мес
- Нет выделенной СБ
- HR хочет автоматизировать

**Secondary:**
- Кадровые агентства
- Enterprise (upsell)
- Банки, финтех
- Госсектор

### Оценка addressable market

| Сегмент | Компаний | Наймов/год | Проверок/год |
|---------|----------|------------|--------------|
| 50-200 чел | ~50k | 10-30 | 500k-1.5M |
| 200-500 чел | ~20k | 30-100 | 600k-2M |
| 500+ чел | ~10k | 100-500 | 1M-5M |
| **Всего** | 80k | — | **~5M проверок/год** |

- 800 клиентов × 50k₽ = **40M₽ ARR**

При 5% конверсии: **200M₽ ARR**

## Конкурентный анализ

### Основные игроки

| Сервис | Фокус | Цена | Сильные стороны | Слабые стороны |
|--------|-------|------|-----------------|----------------|
| **CheckPerson** | Физлица, HR | 150₽/проверка, пакеты от 9.7k₽ | Быстро, удобно, API | Только пакеты |
| **SpectrumData** | Комплекс B2B | Индивидуально | Экосистема, API | Enterprise-цены |
| **СПАРК** | Юрлица + физлица | ~25k₽/мес | Полнота данных | Очень дорого |
| **Контур.Фокус** | Юрлица | 50-70k₽/год | Бренд, доверие | Плохо физлиц |
| **Saby (СБИС)** | Комплекс | Enterprise | 14 параметров | Для крупных |

### CheckPerson — главный прямой конкурент

**Профиль:**
- Часть экосистемы SpectrumData
- Работает с 2017 года
- API + личный кабинет + мобильное приложение

**Тарифы:**

| Пакет | Проверок | Цена | За проверку |
|-------|----------|------|-------------|
| Персонал | 50 | 9,750₽ | 195₽ |
| Персонал | 100 | 18,000₽ | 180₽ |
| Персонал | 200 | 33,000₽ | 165₽ |
| Иностранцы | 50 | 5,500₽ | 110₽ |
| Водители | 50 | 9,750₽ | 195₽ |

**Отзывы:**

Положительные:
> "CheckPerson для нас незаменимая вещь. Каждый раз при найме пользуемся."

> "Время проверки сократилось с 3 дней до 1 часа."

Отрицательные:
> "Нельзя заказать одну проверку, нужно сразу покупать пакет."

> "Денег дерут некисло, иногда выдаёт что информации недостаточно, но деньги уже сняты."

### СПАРК (Интерфакс) — enterprise

**Цены:** ~25,000₽/мес (минимум 3 мес)

**Отзывы:**

Положительные:
> "Качество сервиса высокое."

Отрицательные:
> "Небольшим организациям сложно потянуть 50-70k на подобный сервис."

### Контур.Фокус

**Цены:** 50-70k₽/год

**Отзывы:**

> "Цена кусается."

> "Плохо ищет арбитражные дела: знаю что у компании 23 дела, а Фокус находит 17."

> "Дорого, сложно, непонятно."

## Стратегия роста

### Phase 1: MVP & Validation (Month 1-4)

**Цель:** 10 платящих клиентов

**Тактики:**
1. Собрать 5-7 источников данных (ФССП, МВД, ФНС, ЕГРЮЛ, суды)
2. Простой web-интерфейс
3. Холодный outreach в HR-комьюнити
4. Free tier: 5 проверок бесплатно

**Ключевые вопросы:**
- Доступность API источников
- Юридическая чистота (ФЗ-152)
- Стоимость данных

### Phase 2: Product-Market Fit (Month 5-8)

**Цель:** 50 клиентов, NPS > 40

**Фокус:**
1. Добавить API для интеграции
2. Найти winning segment (HR vs СБ vs агентства)
3. Case studies
4. Referral program

### Phase 3: Growth (Month 9-12)

**Цель:** 100 клиентов

**Каналы:**
1. **SEO** — "проверка кандидатов онлайн", "проверка сотрудников API"
2. **Контент** — guides, чек-листы для HR
3. **Партнёрства** — ATS (HuntFlow, Potok), HR-платформы
4. **Ивенты** — HR-конференции

### Phase 4: Scale (Year 2+)

1. **Интеграции** — ATS, 1C:ЗУП, HRM системы
2. **Vertical expansion** — финтех, гиг-экономика
3. **Geographic** — СНГ (Казахстан, Беларусь)
4. **Enterprise** — кастомные решения

## Честная оценка: стоит ли?

### Аргументы ЗА

1. **Burning problem** — HR тратят 2-3 часа на проверку
2. **Compliance-driven** — банки, госсектор ОБЯЗАНЫ проверять
3. **Sticky** — интегрируется в hiring flow
4. **High ARPU** — 50k₽/год не вопрос для компании
5. **Clear ROI** — 99₽ vs 2 часа работы HR
6. **Растущий рынок** — кадровый голод, удалёнка

### Аргументы ПРОТИВ

1. **Сильный конкурент** — CheckPerson уже есть, работает, дёшево
2. **Зависимость от источников** — API могут закрыть
3. **Юридические риски** — ФЗ-152, персданные
4. **B2B sales** — нужны продажники, длинный цикл
5. **Не mass market** — нужны компании 50+ чел

### Вердикт

**Лучше чем Inventory SMB, но CheckPerson — серьёзный конкурент.**

Возможные стратегии:
1. **Нишевание** — только водители, только финтех, только гиг-экономика
2. **Интеграции** — first-class ATS integration (HuntFlow, Potok)
3. **Дешевле** — агрессивный pricing для SMB
4. **Больше данных** — добавить что-то, чего нет у CheckPerson

**Ключевой вопрос:** Можете ли вы получить доступ к тем же источникам данных дешевле/лучше?
