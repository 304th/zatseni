# Telegram Analytics (TGStat/Telemetr Clone): Deep Dive

Сервис аналитики Telegram-каналов — альтернатива TGStat и Telemetr

---

## Суть продукта

### Проблема

Владельцы каналов, рекламодатели и SMM-специалисты хотят:
- Анализировать статистику каналов
- Находить площадки для рекламы
- Проверять накрутку
- Следить за конкурентами
- Оценивать эффективность рекламы

**Существующие решения:**
- **TGStat** — лидер, бесплатный базовый доступ
- **Telemetr** — платный, более детальный
- **Встроенная аналитика Telegram** — базовая

### Решение

Еще один сервис аналитики? Нужна уникальная фишка:

| Возможный фокус | Описание |
|-----------------|----------|
| **Дешевле** | Демпинг цен |
| **Лучше API** | Для разработчиков и автоматизации |
| **Нишевый** | Только для рекламодателей / только для авторов |
| **Real-time** | Мгновенные обновления (у TGStat — раз в сутки) |
| **AI-анализ** | Предсказание роста, оценка качества контента |
| **Интеграции** | С CRM, рекламными кабинетами |

## Конкурентный анализ

### TGStat — доминирующий игрок

**Профиль:**
- Крупнейший каталог в России
- 17M+ пользователей в месяц
- Бесплатный базовый доступ
- Работает с 2017-2018

**Трафик (SimilarWeb):**
- tgstat.com + tgstat.ru — десятки миллионов визитов
- Главный конкурент telemetr.io — 6.7M визитов

**Тарифы:**

| Тариф | Цена | Особенности |
|-------|------|-------------|
| Бесплатный | 0₽ | Базовая аналитика |
| Premium-аналитика | от 299₽/мес | Без ограничений |
| Premium-поиск | от 500₽/мес | Поиск по публикациям |
| Корпоративный | Индивидуально | Несколько аккаунтов |
| API | Индивидуально | Для разработчиков |

**Сильные стороны:**
- Бренд, доверие
- Бесплатный доступ привлекает массы
- Огромная база данных
- API
- Рекламная площадка

**Слабые стороны:**
- Обновление раз в сутки (не real-time)
- Нет ERR (вовлечённость) в боте
- Рекламный сервис с негативными отзывами
- Баги с обновлением

### Telemetr — платный конкурент

**Профиль:**
- Фокус на детальную аналитику
- База: 9.4M+ каналов
- Данные с 2017 года
- Метки накруток

**Тарифы:**

| Тариф | Цена |
|-------|------|
| Демо | Бесплатно (15 каналов, ограничения) |
| Подписка 1 мес | 2,100-5,000₽ |
| Подписка 6 мес | 7,500₽ (экономия) |

**Сильные стороны:**
- Глубокая аналитика
- Детекция накруток
- Мониторинг упоминаний
- Анализ источников трафика

**Слабые стороны:**
- Нет бесплатного доступа (только демо)
- Цены выросли кратно
- Нет аналитики сторис
- Не все чаты отображаются
- Техподдержка только днём

## Что не хватает рынку (возможности)

### Проблемы текущих сервисов

| Проблема | TGStat | Telemetr |
|----------|--------|----------|
| Обновление раз в сутки | ✓ | — |
| Нет ERR в боте | ✓ | — |
| Нет аналитики сторис | — | ✓ |
| Нет внешних источников | — | ✓ |
| Чаты не отображаются | — | ✓ |
| Высокие цены | — | ✓ |
| Нет freemium | — | ✓ |

### Возможные ниши

1. **Real-time аналитика** — мгновенные обновления, alerts
2. **Stories аналитика** — никто не делает хорошо
3. **AI-скоринг качества** — предсказание роста канала
4. **Fraud detection** — лучшее определение накруток
5. **Рекламный ROI** — трекинг эффективности размещений
6. **API-first** — для разработчиков, интеграций
7. **Чаты/группы** — фокус на комьюнити, не только каналы

## Честная оценка: стоит ли?

### Аргументы ПРОТИВ (много)

1. **TGStat — монополист с бесплатным доступом**
   - 17M пользователей в месяц
   - Бесплатный базовый функционал
   - Как конкурировать с бесплатным?

2. **Высокие барьеры входа**
   - Нужно собрать базу миллионов каналов
   - Инфраструктура для парсинга
   - Юридические риски (ToS Telegram)

3. **Низкая маржа**
   - Цены 300-2000₽/мес
   - Высокие инфраструктурные расходы
   - Нужно много клиентов для прибыли

4. **Telegram может всё сломать**
   - Ограничить API
   - Сделать свою аналитику лучше
   - Изменить правила

5. **Winner-takes-all рынок**
   - Сетевые эффекты: все идут туда, где больше данных
   - TGStat уже выиграл

### Аргументы ЗА (немного)

1. **Рынок растёт** — реклама в TG +54% YoY
2. **Telemetr доказал** — можно быть #2 с платной моделью
3. **Ниши есть** — real-time, stories, AI
4. **B2B потенциал** — агентства, бренды платят больше

## Финансовая модель (если делать)

### Assumptions

| Параметр | Значение |
|----------|----------|
| TAM (рекламодатели + авторы) | ~100k компаний/авторов |
| Платят за аналитику | ~5-10% |
| Addressable | ~5-10k клиентов |

### Pricing

| Тариф | Цена | Целевая аудитория |
|-------|------|-------------------|
| Бесплатный | 0₽ | Привлечение |
| Pro | 990₽/мес | Авторы каналов |
| Agency | 2,990₽/мес | SMM-агентства |
| Enterprise | 9,990₽/мес | Бренды |
| API | от 5,000₽/мес | Разработчики |

### Unit Economics

| Метрика | Значение |
|---------|----------|
| ARPU | 15,000₽/год |
| Gross margin | 60% (высокие инфра-расходы) |
| CAC | 10,000₽ |
| LTV (18 мес) | 22,500₽ |
| LTV:CAC | 2.25:1 |

**Проблема:** LTV:CAC слабый для SaaS (нужно 3:1+)

### Финансовые проекции

| Период | Клиенты | MRR | ARR | Gross Profit |
|--------|---------|-----|-----|--------------|
| Year 1 | 100 | 1.5M₽ | 18M₽ | 900k₽/mo |
| Year 2 | 300 | 4.5M₽ | 54M₽ | 2.7M₽/mo |
| Year 3 | 600 | 9M₽ | 108M₽ | 5.4M₽/mo |

**3-year profit: ~50M₽**

**НО:** Это оптимистичный сценарий. Конкуренция с бесплатным TGStat делает его маловероятным.

## Реалистичный сценарий

### Вероятности

| Сценарий | Вероятность | Результат |
|----------|-------------|-----------|
| Не взлетит (TGStat убьёт) | 60% | Потеря времени/денег |
| Нишевой игрок (500 клиентов) | 30% | 20-30M₽ за 3 года |
| Успех (#2 после TGStat) | 10% | 100M₽+ за 3 года |

### Expected Value

`0.6 × (-5M₽) + 0.3 × 25M₽ + 0.1 × 100M₽ = -3M + 7.5M + 10M = 14.5M₽`

**Expected value положительный, но:**
- Высокий риск полной потери
- Много работы
- Нужна уникальная фишка

## Что нужно для успеха

### Обязательные условия

1. **Уникальная фишка** — нельзя быть просто "ещё одним TGStat"
2. **Инфраструктура** — серверы, парсинг, хранение терабайтов
3. **Юридическая защита** — ToS Telegram, персданные
4. **Freemium** — без бесплатного доступа не привлечь массу

### Возможные стратегии

| Стратегия | Описание | Риск |
|-----------|----------|------|
| **API-first** | Лучший API для разработчиков | Низкий TAM |
| **Real-time** | Мгновенные обновления + alerts | Дорогая инфра |
| **AI-скоринг** | Предсказание роста, качество | Нужна модель |
| **Fraud focus** | Лучшая детекция накруток | Узкая ниша |
| **B2B only** | Только агентства и бренды | Длинный sales |
| **White-label** | Для рекламных платформ | Зависимость |

## Вердикт

### НЕ рекомендую

**Причины:**
1. TGStat — бесплатный монополист, убьёт любого конкурента
2. Высокие инфраструктурные расходы
3. Юридические риски (ToS Telegram)
4. Слабая unit economics
5. Winner-takes-all рынок

### Когда имеет смысл

- Если есть **уникальная технология** (AI, real-time инфра)
- Если есть **captive audience** (свой рекламный бизнес в TG)
- Если делать как **side project** с минимальными инвестициями
- Если фокус на **B2B/API** для интеграций

### Альтернативы лучше

Из всех рассмотренных идей, **Telegram Analytics — худший выбор**:

| Идея | Рекомендация | Почему |
|------|--------------|--------|
| Compliance | ★★★★☆ | Timing, страх, нет freemium-конкурентов |
| Background Checks | ★★★☆☆ | Burning problem, high WTP |
| Inventory SMB | ★★☆☆☆ | Low WTP, hard to reach |
| **Telegram Analytics** | ★☆☆☆☆ | Бесплатный монополист |

## Если всё-таки делать

### MVP (минимум)

1. Парсинг 100k каналов
2. Базовая аналитика (подписчики, посты, ERR)
3. Поиск по каналам
4. Бесплатный доступ (ограниченный)
5. Подписка 990₽/мес

**Срок:** 2-3 месяца
**Бюджет:** 500k-1M₽ (серверы, разработка)

### Фокус для дифференциации

Выбрать ОДНУ фишку и сделать лучше всех:
- Real-time alerts ("канал начал терять подписчиков")
- AI-скоринг ("этот канал накручен на 40%")
- Stories аналитика
- Чаты/группы вместо каналов
