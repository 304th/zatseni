Платформа обучения киберосведомленности — альтернатива KnowBe4, StopPhish, Phishman

---

## Почему это лучше Compliance?

| Фактор | Compliance (ФЗ-152) | Security Awareness |
|--------|---------------------|-------------------|
| Рынок | ~10B₽ (152-ФЗ ниша) | 3B₽ растущий до 10B₽+ |
| Конкуренция | Б-152 (14 лет), много игроков | Меньше игроков, KnowBe4 ушёл |
| Регуляторное давление | Штрафы с мая 2025 | ФСТЭК №117 с марта 2026 |
| Unit economics | One-time setup + мониторинг | Per-user recurring (100-300₽/мес) |
| Масштабируемость | Нужна экспертиза юристов | Чистый SaaS, автоматизация |
| Retention | Средний (после compliance — уходят) | Высокий (ежегодное обучение) |
| Технический moat | Низкий (шаблоны документов) | Высокий (ML, phishing detection) |

**Вердикт: Security Awareness выигрывает по масштабируемости и retention**

## Суть продукта

### Проблема

> "Более 90% всех угроз ИБ — социальная инженерия" — Avast Q1/2024

**Для компаний:**
- Сотрудники — слабое звено в безопасности
- Фишинг атаки растут каждый год
- Регуляторы требуют обучения (ФСТЭК №117)
- Утечки данных = штрафы + репутация
- Западные инструменты (KnowBe4) ушли из России

**Текущие решения:**
- Разовые тренинги (неэффективно)
- Excel-таблицы с "пройденными курсами"
- Дорогие enterprise-решения
- Отсутствие практики (только теория)

### Решение

```
Автоматизированная платформа:
  1. Симуляция фишинговых атак →
  2. Обучающие курсы по результатам →
  3. Трекинг прогресса сотрудников →
  4. Отчёты для compliance
```

### Ключевые функции

1. **Phishing Simulation** — имитация атак (email, SMS, QR)
2. **Adaptive Learning** — курсы по уязвимостям сотрудника
3. **Gamification** — геймификация, соревнования между отделами
4. **Compliance Reports** — отчёты для ФСТЭК, аудиторов
5. **Russian Content** — контент на русском, локальные сценарии
6. **LMS Integration** — интеграция с корпоративными системами

## Бизнес-модель

### Pricing (предложение)

| Тариф | Цена | Особенности |
|-------|------|-------------|
| **Free** | 0₽ | До 10 сотрудников, базовые курсы |
| **Starter** | 99₽/user/мес | До 50 сотрудников, phishing sim |
| **Pro** | 199₽/user/мес | До 500 сотрудников, full features |
| **Enterprise** | 149₽/user/мес | 500+, SLA, интеграции |

**Сравнение с рынком:**
- KnowBe4: $1.70-2.65 = 170-265₽/user/мес
- Российские: цены скрыты, но ~200-500₽/user/мес (enterprise)
- Наша цель: 100-200₽/user/мес (дешевле на 30-50%)

### Unit Economics

| Метрика | Значение |
|---------|----------|
| ARPU (avg company) | 50 users × 150₽ = 7,500₽/мес |
| Annual contract | 90,000₽/год |
| Gross margin | 85% (SaaS, минимум costs) |
| CAC | 30,000₽ |
| LTV (3 years) | 270,000₽ × 85% = 229,500₽ |
| LTV:CAC | **7.6:1** ✅ |

**Отличная unit economics! Лучше чем Compliance (2.5:1)**

### Revenue Streams

1. **Subscriptions** — 80% revenue
2. **Custom Content** — создание сценариев под клиента
3. **Compliance Audit** — отчёты для регуляторов
4. **API/White-label** — для интеграторов ИБ

## Вердикт

### ★★★★★ Лучшая идея из рассмотренных

**Почему лучше Compliance:**

1. **Scalability** — чистый SaaS без юристов
2. **Unit economics** — LTV:CAC 7.6:1 vs 2.5:1
3. **Retention** — ежегодное обучение vs one-time compliance
4. **Technical moat** — ML/AI создаёт защиту
5. **Exit potential** — выше (SaaS multiple 5-10x ARR)

**Почему стоит делать:**

1. KnowBe4 ушёл — timing идеальный
2. ФСТЭК №117 — regulatory tailwind
3. Спрос +20% — fastest growing в ИБ
4. Конкуренты enterprise-only — SMB недообслужен
5. Freemium отсутствует — возможность захвата

### Когда НЕ делать

1. Нет опыта в ИБ-продажах
2. Не готовы инвестировать в контент (курсы, сценарии)
3. Нет терпения на B2B sales cycle
